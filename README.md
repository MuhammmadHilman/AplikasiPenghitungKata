# Aplikasi Penghitung Kata

Tugas 5 - Aplikasi Penghitung Kata

# Deskripsi
Aplikasi Penghitung Kata adalah program berbasis GUI yang dikembangkan menggunakan Java dan Swing di NetBeans. Aplikasi ini bertujuan untuk menghitung jumlah kata, karakter, kalimat, dan paragraf dari teks yang diinputkan oleh pengguna. Selain itu, aplikasi ini memiliki fitur tambahan, seperti pencarian kata tertentu dalam teks serta opsi untuk menyimpan hasil perhitungan ke dalam file.

# Features

Fitur tambahan dari aplikasi ini meliputi:

1. Hitung Jumlah Kata dan Karakter: Aplikasi menghitung jumlah kata dan karakter dari teks yang dimasukkan oleh pengguna.
2. Hitung Jumlah Kalimat dan Paragraf: Aplikasi juga dapat menghitung jumlah kalimat dan paragraf untuk analisis lebih mendalam.
3. Pencarian Kata Tertentu: Fitur ini memungkinkan pengguna mencari kata tertentu dalam teks dan melihat frekuensinya.
4. Simpan Hasil ke File: Pengguna dapat menyimpan teks beserta hasil perhitungannya ke dalam file eksternal untuk referensi di masa mendatang.

# Komponen GUI

Aplikasi ini menggunakan komponen berikut:

- JFrame
- JPanel
- JTextArea untuk input teks dalam JScrollPane
- JLabel untuk menampilkan hasil perhitungan
- JButton untuk memicu perhitungan dan menyimpan hasil

# Logika Program

Aplikasi memanfaatkan manipulasi string dan ekspresi reguler untuk melakukan perhitungan jumlah kata, karakter, kalimat, dan paragraf secara akurat.

    private void hitungTeks() {

    private void btnCariKataActionPerformed(java.awt.event.ActionEvent evt) {                                            
        // TODO add your handling code here:
        String teks = txtAreaInput.getText();
        String kataCari = txtCariKata.getText();
    
        int jumlahKataDitemukan = teks.split("(?i)\\b" + kataCari + "\\b").length - 1;
    
        JOptionPane.showMessageDialog(this, "Jumlah kemunculan kata '" + kataCari + "': " + jumlahKataDitemukan);
    }                                           

    private void btnSimpanActionPerformed(java.awt.event.ActionEvent evt) {                                          
        // TODO add your handling code here:
        try {
            JFileChooser fileChooser = new JFileChooser();
            if (fileChooser.showSaveDialog(this) == JFileChooser.APPROVE_OPTION) {
                File file = fileChooser.getSelectedFile();
            
            try (PrintWriter writer = new PrintWriter(file)) {
                writer.println("Teks:");
                writer.println(txtAreaInput.getText());
                writer.println();
                writer.println("Hasil Perhitungan:");
                writer.println("Jumlah Karakter: " + lblJumlahKarakter.getText());
                writer.println("Jumlah Kata: " + lblJumlahKata.getText());
                writer.println("Jumlah Kalimat: " + lblJumlahKalimat.getText());
                writer.println("Jumlah Paragraf: " + lblJumlahParagraf.getText());
                writer.flush();
            }   
            
                JOptionPane.showMessageDialog(this, "Hasil perhitungan berhasil disimpan.");
            }
        } catch (Exception e) {
            JOptionPane.showMessageDialog(this, "Terjadi kesalahan saat menyimpan file.");
        }
    }
